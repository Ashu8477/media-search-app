# 🔎 Media Search App

A modern media search application built with **React**, **Redux Toolkit**, and **Vite**. Search for **Photos**, **Videos**, and **GIFs** from multiple public APIs and save your favorite media into personal collections.

## 🌐 Live Demo

👉 https://media-search-app-jade.vercel.app/

---

## ✨ Features

- 🔍 Search media with a single search bar
- 📷 Browse high-quality photos
- 🎥 Search videos
- 🎭 Find trending and relevant GIFs
- ❤️ Save favorite items into collections
- 🗂 View saved collections anytime
- ⚡ Global state management using Redux Toolkit
- 📱 Responsive UI

---

## 🛠 Tech Stack

- React
- Redux Toolkit
- React Router DOM
- Axios
- Tailwind CSS
- Vite

---

## 🔗 APIs Used

- Unsplash API (Photos)
- Pexels API (Videos)
- Giphy API (GIFs)

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Ashu8477/media-search-app.git
```

### Install dependencies

```bash
npm install
```

### Create a `.env` file

```env
VITE_UNSPLASH_KEY=YOUR_UNSPLASH_API_KEY
VITE_PIXEL_KEY=YOUR_PEXELS_API_KEY
VITE_GIPHY_KEY=YOUR_GIPHY_API_KEY
```

### Run the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```
